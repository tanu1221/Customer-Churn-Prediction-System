# Customer Churn Prediction System

Machine Learning based Customer Churn Prediction System with Streamlit Web App and Power BI Dashboard.

## Project Overview

This project predicts whether a telecom customer is likely to churn based on customer demographics, account information, and service usage patterns.

The system uses Machine Learning to identify customers at risk of leaving and provides interactive visualizations through Streamlit and Power BI dashboards.

## Features

- Customer Churn Prediction
- Interactive Streamlit Dashboard
- Customer Analytics and Insights
- Power BI Dashboard
- XGBoost Machine Learning Model
- Real-time Prediction Interface

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- XGBoost
- Streamlit
- Plotly
- Power BI
- Joblib

## Machine Learning Model

Model Used: XGBoost Classifier

Model Accuracy: 79%

## Dataset

Dataset: Telco Customer Churn Dataset

Total Customers: 7043

Churn Rate: 26.5%
